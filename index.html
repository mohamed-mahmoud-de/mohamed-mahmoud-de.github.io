03
01 —
NileFlow — Traffic Intelligence Platform DEPI Capstone Team Lead

Leading a 6-member team building a real-time congestion monitoring platform for Greater Cairo and Alexandria. Ingesting live traffic (TomTom API), weather (Open-Meteo), and transit position events through three Kafka topics. Processing with Spark Structured Streaming using time-window joins to compute a rolling congestion index per corridor and detect anomalies. Writing to Cassandra for time-series metrics and Elasticsearch for searchable alert history. Airflow DAGs for daily baseline recalculation and data quality checks. Redis pub/sub for anomaly fan-out and Discord webhook alerts visualized on a live Streamlit map dashboard.

Apache Kafka Apache Spark Apache Airflow Cassandra Elasticsearch Redis PostgreSQL Docker Streamlit
↗
02 —
Real-Time Crypto Trade Pipeline Shipped

End-to-end streaming pipeline ingesting live cryptocurrency trades from Binance WebSocket, publishing to Kafka, and processing with Spark Structured Streaming using 1-minute tumbling window aggregations and 2-minute watermarks for late event tolerance. Dual-sink writes: Cassandra (partitioned by symbol, clustered by window start) for low-latency time-range lookups, and Elasticsearch for inverted-index search across any field. Visualized through Kibana dashboards. Resolved Spark 4.x Elasticsearch connector incompatibility by unifying both sinks into a single foreachBatch function.

Apache Kafka Apache Spark Cassandra Elasticsearch Kibana Docker

03 —
Weather Data Pipeline Personal Project

An automated ETL pipeline that extracts hourly weather data for Egyptian cities from a public API, transforms it with Python, and loads into PostgreSQL using a schema designed for time-series queries and historical comparisons. Orchestrated in an Airflow DAG with scheduled runs, retry logic, structured logging, and Discord webhook alerting. Containerized with Docker Compose; designed for multi-city extensibility and historical backfill.

Python Apache Airflow PostgreSQL Docker ETL

04 —
Egypt Jobs Pipeline V3 Complete

A three-version data pipeline scraping Egyptian tech job listings from Wuzzuf. V1: naive Python script. V2: full Apache Airflow DAG with 2-hour scheduling, exponential backoff retries, Discord webhook alerting, backfill support, and a Streamlit dashboard. V3: a real-time alert layer — crash-proof polling loop checking Wuzzuf every 3 minutes, Redis SET deduplication for sub-millisecond seen-job lookups, keyword filters by role and seniority, and instant Discord notifications on new listings. Each version was built by hitting the wall the previous one couldn't solve.

Python Apache Airflow PostgreSQL Redis Streamlit Docker BeautifulSoup ETL
