Weather Data Pipeline

Designed and built an end-to-end weather data engineering pipeline that ingests daily weather data from a public API, processes and transforms it using Python, and stores it in a cloud-based data warehouse following a star schema model. Orchestrated workflows with Apache Airflow, implemented data quality checks, and optimized transformations using dbt for incremental processing. The pipeline feeds an interactive Power BI dashboard enabling trend analysis, temperature forecasting insights, and city-level comparisons.

02 —
Real-Time Streaming Data Pipeline

Developed a real-time stock market data streaming platform using Apache Kafka and Spark Structured Streaming to process high-frequency financial data. Implemented window-based aggregations, moving averages, and anomaly detection logic to generate near real-time insights. Stored processed data in Cassandra and Elasticsearch to power a live analytics dashboard in Kibana. Containerized the entire architecture using Docker to simulate a production-ready distributed system.
